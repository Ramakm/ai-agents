Machine learning is a branch of artificial intelligence that enables computers to learn from data and make decisions without being explicitly programmed for every situation. Instead of following hand-coded rules, machine learning systems identify patterns in training data and use those patterns to make predictions or decisions on new, unseen data. The field draws on statistics, linear algebra, calculus, and computer science.

Supervised learning is the most common type of machine learning. In supervised learning, a model is trained on labeled examples, meaning each input in the training dataset is paired with the correct output. The model learns a mapping from inputs to outputs by minimizing the difference between its predictions and the true labels. Common supervised learning tasks include classification, where the output is a category, and regression, where the output is a continuous number. Algorithms used for supervised learning include linear regression, logistic regression, decision trees, random forests, support vector machines, and neural networks.

Unsupervised learning deals with unlabeled data. The model must discover structure in the data on its own without guidance from known outputs. Clustering algorithms group similar data points together. K-means clustering assigns each data point to one of k clusters based on distance from cluster centers. Hierarchical clustering builds a tree of nested clusters. Dimensionality reduction techniques such as principal component analysis compress high-dimensional data into fewer dimensions while preserving important patterns.

Reinforcement learning trains an agent to make sequences of decisions by rewarding desired behaviors and penalizing undesired ones. The agent interacts with an environment, takes actions, and receives rewards or penalties based on the outcomes. Over many iterations, the agent learns a policy that maximizes cumulative reward. Reinforcement learning has achieved notable successes in game playing, robotics, and recommendation systems.

Neural networks are computing systems loosely inspired by the structure of biological brains. A neural network consists of layers of interconnected nodes called neurons. Each connection has a weight that determines how strongly signals pass between neurons. During training, the network adjusts these weights using a process called backpropagation combined with gradient descent optimization. The input layer receives raw data, hidden layers extract increasingly abstract features, and the output layer produces predictions.

Deep learning refers to neural networks with many hidden layers, sometimes numbering in the hundreds. Deep networks can learn complex, hierarchical representations from raw data such as images, audio, and text. Convolutional neural networks excel at image recognition by applying filters that detect edges, textures, and shapes at different scales. Recurrent neural networks process sequences by maintaining hidden states that capture information from previous inputs.

Overfitting occurs when a model learns the training data too precisely, including its noise and random variation, and performs poorly on new data. Techniques to reduce overfitting include regularization methods such as L1 and L2 penalties that discourage large weights, dropout which randomly disables neurons during training, and early stopping which halts training before the model memorizes the training set. Cross-validation assesses model performance by training and evaluating on different splits of the data.

Feature engineering is the process of selecting and transforming raw input variables to improve model performance. Good features capture the underlying structure of the problem. For tabular data, useful transformations include normalization, standardization, log transformation, and encoding categorical variables. For text data, features might include word counts, term frequency-inverse document frequency scores, or dense vector embeddings. Deep learning partially automates feature engineering by learning representations directly from raw inputs.

The transformer architecture, introduced in 2017, has become the dominant approach for natural language processing tasks. Transformers use a mechanism called self-attention to weigh the importance of different words in a sequence when encoding each word's meaning. This allows the model to capture long-range dependencies in text without the sequential bottleneck of recurrent networks. Large language models such as GPT and BERT are built on transformer architectures and trained on vast amounts of text data.

Model evaluation requires careful selection of performance metrics appropriate to the task. For classification, common metrics include accuracy, precision, recall, F1 score, and area under the receiver operating characteristic curve. For regression tasks, metrics include mean absolute error, mean squared error, and R-squared. In problems with imbalanced classes, accuracy alone can be misleading because a model predicting only the majority class can score high while providing no useful discrimination.
